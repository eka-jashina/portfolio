Сайт проектной компании Евклид.
Учебный проект.
Адаптивная верстка под 4 брейкпоинта.
Используется flexbox.
С помощью js реализованы бургер-меню, поиск, слайдер, аккордион, табы.
Доступна навигация по сайту с клавиатуры и с помощью скринридера. 
Проведена оптимизация шрифтов и изображений, минификация js и css.
Используется методология БЭМ.
