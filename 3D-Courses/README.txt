Сайт продажи курсов 3D-моделирования.
Учебный проект.
В верстке использован Bootstap 4.
Вёрстка под три брейкпоинта. "Минорные" брейкпоинты выбраны для корректного отображения сайта в наибольшем количестве разрешений. (При использовании стандартных сайт адаптировался хуже).
Изображения адаптированы подо все разрешения экрана, уменьшен вес, используется формат webpю.
Используется методология БЭМ.
Используются автопрефиксы.
